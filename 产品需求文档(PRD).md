# 产品需求文档 (PRD) - 记忆卡片训练系统

## 1. 产品概述
**产品名称**：记忆卡片 (JiyiCard)
**产品定位**：一款面向儿童和成人的脑力、记忆力与视觉注意力训练工具。
**目标用户**：需要提升记忆力和专注力的儿童、需要保持大脑活跃的成人/老人。
**产品形态**：响应式 H5 网页应用（可无缝嵌入微信小程序作为 Web-view 使用）。

## 2. 核心功能模块

### 2.1 游戏训练模块
系统包含三大独立的游戏训练模块，每个模块具有独立的关卡系统和难度递增机制。

#### 2.1.1 闪卡记忆训练 (Game 1)
- **玩法**：在极短时间内（如1-3秒）展示 N 张目标卡片（动物Emoji），随后隐藏目标并混入干扰项，玩家需从多个选项中找出刚才展示过的所有目标卡片。
- **机制**：答错任何一张即判负；全部选对则进入下一局；达到指定胜局数（如5局）后解锁下一关卡。
- **配置维度**：目标卡片数量、展示时间、总选项数量（干扰项数量）。

#### 2.1.2 连续记忆训练 (Game 2)
- **玩法**：“无尽流”挑战模式。初始展示 N 张目标卡片让用户记忆。进入选择阶段后，系统会在原卡片中混入 N 张“新卡片”，用户必须且只能点击这 N 张“新卡片”。
- **机制**：每次成功点击新卡片后，系统立即追加新卡片并重新洗牌（困难模式）或在原位生成（普通模式），挑战用户的短时记忆极限。点错旧卡片即游戏结束。
- **模式选择**：
  - **普通模式**：已出现的卡片位置固定不变。
  - **困难模式**：每次点击后，所有卡片位置随机打乱。
- **配置维度**：每次新增卡片数量、初始记忆时间。

#### 2.1.3 视觉注意力训练 (Game 3)
- **玩法**：短暂展示由不同形状（圆形、方形、星形等）和不同颜色（红、蓝、绿等）组合而成的几何图形。随后混入颜色或形状相似的干扰项，要求玩家准确找出目标图形。
- **模式选择**：
  - **普通模式**：无顺序要求，选出所有目标图形即可。
  - **困难模式**：必须严格按照目标图形出现的先后顺序进行选择，顺序错误即判负。
- **配置维度**：目标图形数量、展示时间、总选项数量。

### 2.2 激励与消耗系统
- **电量机制 (体力值)**：每次开始任意一局训练都会消耗 1 点电量。电量耗尽无法开始游戏。
- **记忆沙漏 (复活道具)**：在 Game 2 或 Game 3 中挑战失败时，可消耗 1 个记忆沙漏获得一次“重新看一次图形/恢复记忆时间”的机会，免除当前轮次的失败惩罚。
- **排行榜与记录**：每个游戏模块独立记录用户的最高得分、用时和最近3次的历史记录（精确到时分秒）。

### 2.3 家长专区 (控制台)
- **儿童锁机制**：进入家长专区必须先解开随机生成的数学加法题，防止儿童误操作。每次进入强制验证。
- **数据看板**：家长可直观查看各训练模块的当前最高等级进度、当前剩余电量和沙漏数量。
- **资源获取**：
  - 看广告/视频补充电量（如看一次+5局）。
  - 看广告/视频获取记忆沙漏（如看一次+1个）。
  - 具有广告冷却时间（CD）机制，防止连续刷资源。
- **系统设置**：音效开关、一键清除缓存等功能。

### 2.4 账号与用户系统
- **多账号支持**：支持用户名/邮箱注册登录。数据基于用户 ID 隔离，确保不同用户在同一设备上数据不互串。
- **快捷登录**：提供“微信一键登录”模拟入口，方便快速测试和体验。

## 3. 非功能需求
- **响应式设计**：完美适配手机端竖屏比例，同时兼容 Pad 和 PC 端。
- **动效体验**：提供平滑的翻牌、洗牌、成功/失败弹窗动画，增强游戏反馈。
- **部署要求**：前后端分离架构，支持云端 Serverless 部署。
