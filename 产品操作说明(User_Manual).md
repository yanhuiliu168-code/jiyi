# 产品操作说明 (User Manual) - 记忆卡片训练系统

## 1. 登录与注册
**访问地址**：打开浏览器或微信，访问您的专属游戏链接。
1. **账号注册**：首次使用，请点击“去注册”，输入用户名、邮箱和密码完成账号创建。
2. **账号登录**：已有账号用户直接输入用户名/邮箱和密码登录。
3. **微信一键登录 (测试专用)**：点击底部的绿色按钮，系统会自动为您生成一个测试账号并瞬间登入游戏，适合快速体验所有功能。

---

## 2. 认识首页界面
进入游戏后，您会看到主控制台：
- **顶部状态栏**：
  - **电量 (🔋)**：显示您当前剩余的训练次数（如 20 局）。每次开始游戏会消耗 1 局。
  - **沙漏 (⏳)**：显示您拥有的“复活道具”数量。
- **三大训练模块**：
  - **闪卡记忆训练**：经典瞬时记忆玩法。
  - **连续记忆训练**：挑战短时记忆极限的无尽模式。
  - **视觉注意力训练**：颜色与形状的复合记忆挑战。
- **家长专区入口**：位于页面最下方，带有一把小锁图标。

---

## 3. 游戏玩法详解

### 3.1 闪卡记忆训练
1. 点击进入后，选择关卡（锁定的关卡需要先通关前置关卡）。
2. 页面会短暂展示 1~5 张动物卡片，请**死死盯住并记住它们**！
3. 卡片消失后，页面会出现一堆混杂了干扰项的卡片。
4. **准确无误地点击刚才出现过的所有卡片**。
5. 只要点错一张，当前回合立刻失败；全部找齐则进入下一回合。完成 5 个回合即通关。

### 3.2 连续记忆训练 (无尽模式)
1. 进入关卡后，首先记住屏幕上出现的 N 张卡片。
2. 倒计时结束后，卡片会重新排列，并**混入 N 张全新的卡片**。
3. 您的任务是：**只点击这 N 张新出现的卡片**。
4. 只要您点对了，系统会立刻再追加 N 张新卡片。您必须在脑海中不断更新“我已经点过哪些旧卡片了”，挑战自己的记忆极限！
5. **普通模式**：卡片位置不动。
6. **困难模式**：每次点击后卡片位置会瞬间打乱！

### 3.3 视觉注意力训练
1. 记住屏幕上短暂闪现的几何图形（包括它的**形状**和**颜色**，例如：红色的五角星、蓝色的圆形）。
2. 在选项中找出它们。
3. 如果您选择了**困难模式**，您不仅要找出它们，还必须**严格按照它们刚才从左到右出现的顺序**进行点击！

---

## 4. 失败与复活
- 在 **连续记忆** 和 **视觉注意力** 训练中，如果您不小心点错了，系统会提示“挑战失败”。
- 如果您拥有**【记忆沙漏】**道具，此时会弹出一个橙色的按钮。
- 点击**“使用沙漏复活”**，系统会消耗 1 个沙漏，并**再给您看一次刚才的图形**，让您继续当前的游戏进度，不用从头开始！

---

## 5. 家长专区 (获取电量与道具)
当电量耗尽或沙漏用完时，请前往【家长专区】补充。
1. **儿童锁验证**：点击首页底部的“家长专区”，系统会要求您计算一道两位数的加法题（如 `12 + 8 = ?`）。输入正确答案后才能进入。
2. **数据查看**：您可以直观地看到孩子的各个游戏等级和历史战绩。
3. **补充资源**：
   - 点击 **“补充电量”**：模拟观看广告后，电量瞬间 +5 局。
   - 点击 **“储备沙漏”**：模拟观看广告后，沙漏数量 +1 个。
   - **注意**：观看后会有冷却时间（如 1 小时），冷却期间按钮会变成灰色。
4. **清除缓存**：如果您想让游戏重头开始，点击底部的红色按钮“清除缓存”，所有账号数据和进度将被重置。

---

## 6. 后台管理 (管理员专用)
如果您是管理员，想要修改游戏的难度和配置：
1. 访问后台管理系统链接。
2. 输入管理员账号 `admin` 和密码 `admin123`。
3. 在面板中，您可以：
   - 修改每局游戏初始赠送的电量。
   - 修改观看广告的冷却时间（秒）。
   - 修改闪卡记忆每关展示的时间和卡片数量。
   - 修改视觉注意力每关的难度。
4. 修改完毕后，点击右上角的 **“保存配置”**，前端游戏会立刻生效。
